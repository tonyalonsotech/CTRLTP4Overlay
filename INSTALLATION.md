## Installation Instructions

1. **Extract Folders:**
   * Move the `DesktopOverlay` folder into: `..\Traktor Pro 4 - Overlay\Resources64\qml\Screens\`
   * Move the `S8` folder into: `..\Traktor Pro 4 - Overlay\Resources64\qml\CSI\`

2. **Configure Display:**
   * Set your desktop resolution strictly to **1920x1080**.

3. **Launch App:**
   * Open Traktor and ensure the application window is **fully maximized**.
