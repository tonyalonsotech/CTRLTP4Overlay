# CTRL TP4 Overlay
An on-screen overlay that displays essential information missing from the Traktor UI.

## Why use this overlay?
- **Missing UI Data:** The Native Instruments Traktor user interface lacks key on-screen information that is otherwise native to hardware controllers like the Traktor Kontrol D2, S5, S8, and S4 MK3.
- **Hardware Independence:** Users without these specialized Traktor controllers can now access critical live data, including FX parameters and Hotcue Names, directly on their monitors.

## Version History

### v1.0
- **Initial Release:** Launched the base overlay application.
- **Resolution Limit:** Currently optimized exclusively for a fullscreen resolution of 1920x1080.
- **FX Limitations:** Restricted to FX Units 1 and 2, and currently only compatible with the Single FX Unit layout. The FX Select dropdown is replaced by previous and next buttons.
